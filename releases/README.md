# Descargas VitalFi (APK firmadas)

APK firmadas listas para instalar en Android 8.0+ (API 26).

## Instalación

1. Descarga la versión que quieras desde la tabla de abajo.
2. Copia el APK al teléfono (USB, WhatsApp, Drive, etc.).
3. Permite **instalar apps de orígenes desconocidos** si Android lo pide.
4. Abre el APK e instala.
5. Concede permiso de **ubicación** al abrir VitalFi (Android lo exige para leer señal Wi-Fi).
